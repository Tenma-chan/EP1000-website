Fusion 360 Assignment 2

Drawing the sketch

This model was drawn according to the dimensions provided in assignment 2. The sketch was drawn using the dimensioning tool, circle, rectangle, inscribed and circumscribed elements. Furthermore the fillet tool was used in order to get the curves at certain parts of the sketch.

Extruding the Sketch

This sketch was then followed up by the Extrude function, a function which adds height or depth to open or closed sketch profiles and faces. This time, we will only be extruding the circular elements.

Extruding the Handle

Next up, we will extrude the handle portion of the wrench. It it important that under "Operation", New Body is selected as this will allow us to get a nice finish on the handle later towards the end.

Fillet the handle

Continuing on, we will be using the Fillet tool to round the corners of the handle, creating a more friendly shape on the handle which will be more comfortable for the user of the tool.

Extruding the Recess

Lastly, we will be making use of the Extrude tool once again in order to create a recess that runs along the side of the handle. When the extrude tool is used into the side of a pre-existing model, the boolean operation will change from a "Join" to a "Cut". This will be used to generate the recess.
